# Gerador de Senhas Aleatórias

## Objetivo
Criar um script que gere senhas aleatórias para uso em sistemas e sites.

## Funcionalidades
- Gerar senhas com letras maiúsculas, minúsculas, números e caracteres especiais.
- Permitir ao usuário escolher o tamanho da senha.

## Cronograma
- **Dia 1:** Configuração inicial e estrutura do código.
- **Dia 2:** Implementação das funcionalidades básicas.
- **Dia 3:** Testes e ajustes finais.
